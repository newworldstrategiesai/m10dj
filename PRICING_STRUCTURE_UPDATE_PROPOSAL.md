## Problems Identified

### 1. **Monogram Was in Package 2 (Error)**
- Wasn't supposed to be there
- Created confusion
- Led to removal request

### 2. **Breakdown Shows Individual Prices**
- Clients see: "4 Hours: $1,600, Ceremony Audio: $500, etc."
- They think: "I can remove items"
- Creates shopping cart mentality

- Admin can remove items
- Client sees breakdown
- No boundary on what can/can't be removed

### 4. **Invoice Structure Confusing**
- Shows line items separately
- Looks like "base price + add-ons"
- Not "complete package"

### Solution 3: **Set Customization Policy**

**Policy:**
1. **Packages are fixed-price bundles**
   - No item removals from standard packages
   - Packages are designed as complete solutions
   - Bundle discount only applies to complete packages

2. **Custom packages available**
   - Priced at a la carte rates (premium)
   - Requires consultation
   - Not the default option

3. **Add-ons only**
   - Clients can ADD items to packages
   - Cannot REMOVE items from packages
   - Add-ons priced at a la carte rates

4. **Exceptions (if any)**
   - Only true visual add-ons can be removed (like monogram)
   - Core services cannot be removed
   - Removals reduce package discount proportionally
