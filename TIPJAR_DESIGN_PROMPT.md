## Product Description

TipJar is a specialized SaaS product designed for event entertainers (primarily DJs) to collect tips and song requests from event attendees in real-time. It's a focused, lightweight alternative to full business management platforms.

### Core Value Proposition

**"Get tipped instantly. Request songs easily. No app downloads required."**

TipJar allows event attendees to:
- Submit song requests via a simple web link
- Send tips/shoutouts with instant payment processing
- Access everything through their phone browser (no app needed)
- Pay with credit cards or Cash App Pay

For DJs/entertainers, TipJar provides:
- Instant payment collection (3.5% + $0.30 platform fee)
- Automatic payouts to bank account (via Stripe Connect)
- Real-time request management
- Simple embeddable widget for websites
- No technical knowledge required
