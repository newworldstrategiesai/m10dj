# OG Image Generation Guide

### TipJar Live

## Tools to Create OG Images

1. **Canva** (Recommended - Easy)
   - Use template: "Facebook Post" or "Social Media Graphic"
   - Size: 1200x630px
   - Export as PNG

2. **Figma** (Professional)
   - Create frame: 1200x630px
   - Design with gradients and logos
   - Export as PNG

3. **Online Tools**
   - https://www.canva.com/
   - https://og-image.vercel.app/ (for text-based OG images)

## Quick Solution

If you need a quick placeholder, you can:
1. Use the existing logos from `/public/assets/`
2. Create a simple gradient background
3. Add text overlay
4. Export as 1200x630px PNG
